# MySQL-learning-practice

This repository contains my **MySQL learning and practice work**.  
It is created to improve my SQL skills through hands-on exercises using different sample databases, tables, and real-world query examples.

The purpose of this repository is to build strong knowledge in **MySQL query writing**, understand database concepts, and practice solving business-style data problems.

---

## 📌 About This Repository

`MySQL-learning-practice` is a personal learning repository where I practice:

- Basic SQL queries
- Filtering and sorting
- Aggregate functions
- Grouping data
- Joins
- Subqueries
- Real-world business queries
- Database table design
- Query logic improvement

This repository may include multiple SQL practice files based on different datasets such as:

- Employees and Departments
- Product and Customer
- Sales and Orders
- Inventory data
- Business reporting examples

---

## 🎯 Learning Goals

The main goals of this repository are:
